## Next.js × Emotion Sample

- Next.js、Emotionで実装した2カラムレイアウトのサンプルになります。

<br><br>

### 使用方法

```bash
# 開発モードで起動する
npm run dev

# 本番モードで起動する
npm run start

# ビルドを行う
npm run build

# storybook画面を起動する
npm run storybook

# ESLint、Prettierによるチェック行う
npm run check

# ESLint、Prettierによるテキスト整形を行う
npm run format
```
